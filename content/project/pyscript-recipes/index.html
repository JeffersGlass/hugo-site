PyScript is a surprisingly simple and relatively comlex entity. At it's core, it allows users to write Python into their HTML with a special kind of script tag. And it does just that! But with that simple idea lies a word of complexity, especially once you realize you can pull in any JavaScript object, make use of Pyodide's extensive API, and start interacting with Web Workers.

That's why I launched pyscript.recipes, a collection of short, rapidly-useful snipets for PyScript to keep developers moving fast.

As of today, the site is fully updated for the newest PyScript release, and live.

The whole site is also open source - have a recipe or correction you'd like to contribute? Feel free to drop in a PR on GitHub.
