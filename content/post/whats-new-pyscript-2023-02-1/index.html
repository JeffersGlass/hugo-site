PyScript

The output attribute of the <py-script> tag has been restored (#1063)

So Long, <py-box>, <py-title>, <py-inputbox>, and <py-button>. As previously promised, these elements (which were deprecated in version 2022.12.1) have been removed in version 2023.02.1.

Runtimes has been renamed interpreter across the board.

JG: Check how this changes external API
(#1082)

Little thing - need an attribute to handle py-[event] syntax, now auto-generated if none is provided (#1122)

Documentation

PyScript now has an incremental Changelog! If you're sick of wading through all ~5000 of my words every time there's a release, the Changelog has the short-and-sweet version (#1066)

The tutorials index page at docs.pyscript.net/tutorials has gotten a facelift, for a better onboarding process for new users (#1090)

Examples

New Pandas example (#1067)

Bug Fixes

Fixed error where minified version of PyScript didn't have access to js module (#1054)

Fixed an error in py-repl where Shift-Enter would insert a newline, in addition to executing the REPL. (#1094)

Fixed an error where exceptions in py-[event] handlers were thrown, but not displayed in the DOM/warning (#1118)

Plugins

Can now fetch JS plugins from URL! (#1065)

Examples inspector plugin (#1040)

Fixed error where Python plugins were being registered with PluginManager twice (#1064)
